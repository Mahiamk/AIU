# Recipe Website Project

## Overview
This project is a simple recipe website created using HTML and CSS. It features a homepage that links to individual recipe pages, each containing detailed instructions and ingredients for various recipes.

## Project Structure
```
Lab_0
├── css
│   └── style.css
├── recipes
│   ├── recipe1.html
│   └── recipe2.html
├── index.html
└── README.md
```

## Files Description

- **css/style.css**: Contains the styles for the website, defining the layout, colors, fonts, and other visual aspects of the HTML elements.

- **recipes/recipe1.html**: HTML structure for the first recipe, including headings, paragraphs, images, and lists to present the recipe details.

- **recipes/recipe2.html**: HTML structure for the second recipe, similar to `recipe1.html`, with its own unique content and layout.

- **index.html**: Serves as the homepage of the website. It includes navigation, an introduction, and links to the individual recipes.

## Instructions for Use
1. Open `index.html` in a web browser to view the homepage.
2. Click on the links to navigate to the individual recipe pages.
3. Customize the recipes by editing the `recipe1.html` and `recipe2.html` files as needed.
4. Modify the styles in `style.css` to change the appearance of the website.
